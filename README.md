## 📌 Project Overview

This repository contains an end-to-end data analytics project based on the **Olist Brazilian E-Commerce Dataset**.

The project was developed using **SQL, Microsoft Excel, Tableau, and Power BI** to analyze customer orders, payment behavior, delivery performance, product categories, and customer review scores.

The same business requirements were analyzed and visualized across multiple analytics tools to demonstrate practical data analysis, dashboard development, and business intelligence skills.

---

## 🎯 Business Objectives

- Compare payment values between weekday and weekend orders
- Analyze orders with a review score of 5 and credit card payments
- Evaluate delivery performance for the Pet Shop category
- Analyze average price and payment values for customers in São Paulo
- Study the relationship between shipping days and customer review scores
- Build interactive dashboards for business analysis and decision-making
